# CE Compliance Agent

Automated continuing education tracking and renewal management for appraisers and home inspectors.

## About

CE Compliance Agent helps real estate appraisers and home inspectors:
- Track CE credits automatically
- Discover state-approved courses
- Receive deadline reminders (90, 60, 30 days)
- Generate board-ready compliance reports
- Stay organized for audits

## Quick Links

- **Website**: [https://webnestam.github.io/CE-Cop-Agent](https://webnestam.github.io/CE-Cop-Agent)
- **Live Preview**: See the product in action on the landing page
- **Pricing**: Solo, Team, and Enterprise plans
- **Free Trial**: 14 days, no credit card required

## Features

### Course Discovery
Find approved courses by state, profession, category, and hours needed.

### Credit Logging
Automatic detection of completion emails and certificate storage.

### Deadline Alerts
Smartly timed reminders with gap analysis and course suggestions.

### Compliance Reports
Generate audit-ready PDFs with certificates attached.

### Team Dashboard
Manage up to 3 licenses with manager oversight and alerts.

## Deployment

This project is built as a static site hosted on GitHub Pages. To deploy:

1. Ensure `index.html`, `logo.png`, and `zelle_qr.jpg` are in the repository root
2. Enable GitHub Pages in repository settings (branch: `main`, folder: `/ (root)`)
3. Site will be live at: `https://webnestam.github.io/CE-Cop-Agent`
